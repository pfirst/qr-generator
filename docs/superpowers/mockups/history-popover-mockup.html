technical · implementation notes

ไฟล์ที่แตะ

  • ลบ src/components/HistoryCard.tsx → สร้าง src/components/HistoryPopover.tsx
  • Header.tsx รับ props recent / onLoad / onClear แล้ว render <HistoryPopover> แทน privacy pill เดิม
  • App.tsx ส่ง props ลง <Header>, ลบ <HistoryCard> ออกจาก grid (cell order-4)

component API

  • HistoryPopover({ recent, onLoad, onClear }) — props เดิมของ HistoryCard เป๊ะ
  • thumbnail ใช้ thumbSvg() เดิม (qrcode + renderQrSvg, async, THUMB style) — ยกมาทั้งก้อน
  • โหลด thumbnails เฉพาะตอน popover เปิด (lazy) เพื่อไม่ render ตอนปิด

behavior

  • trigger: icon + "ประวัติ" + count badge · โชว์ทุกขนาดจอ (เอา hidden md:flex ออก)
  • หัว panel = icon อย่างเดียว: history icon (ซ้าย) + trash icon button (ขวา = ล้าง)
  • list แนวตั้ง · max-height ~344px แล้ว scroll · กดแถว → onLoad(r) + ปิด
  • ปิดเมื่อ: คลิกนอก popover (mousedown listener) หรือกด Escape
  • count = 0 → แสดง empty state, badge ซ่อน

grid reflow

เอา <HistoryCard> (col-1 / row-2) ออก → คอลซ้ายเหลือ DataCard อย่างเดียว, คอลขวา = QrPanel + CheckExport flow ตามความสูงปกติ ลบ class lg:row-start-2 ของ CheckExport ให้ไหลขึ้นเอง
